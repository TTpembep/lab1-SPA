# LAB1 - Single Page Apllication
В результате выполнения проекта было разработано одностраничное веб-приложение, на тему: «Безопасность автомобильного транспорта», с учётом аспектов безопасности.
Для начала была разработана структура базы данных и описаны необходимые таблицы.
Создан REST API на языке программирования Python с использованием фреймворка Django. Реализованы стандартные HTTP-методы: GET, POST, PUT, DELETE, для обеспечения доступа к данным. Также реализованы методы аутентификации и авторизации с использованием JWT-токенов, при помощи библиотеки «SimpleJWT».
Затем применена JavaScript-библиотека – React, для создания фронтенда приложения. По средствам взаимодействия с API, реализован функция авторизации и регистрации пользователей, отображения данных, а также их обновление, удаление и хранение.
Была обеспечена безопасность бэкенда и фронтенда, согласно техническому заданию.
Разработанное приложение позволяет диспетчеру централизованно отслеживать маршруты поездок.
При отклонении водителя с маршрута, диспетчер сможет предпринять необходимые меры по устранению чрезвычайной ситуации.
Водители могут создавать и изменять маршруты, а пассажиры их просматривать.
Данный проект на практике показывает знания, полученные в ходе обучения.
